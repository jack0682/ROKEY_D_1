# 3일 집중 ML급 정밀도 로봇 액체 따르기 프로젝트 - 완전 구체화 계획서

## 🎯 프로젝트 철학 및 목표

### 프로젝트 정체성
이는 단순한 압축 프로젝트가 아니라, **3일이라는 시간 제약 내에서 최고의 성과를 달성하기 위해 설계된 집중형 혁신 프로젝트**다. 모든 요소가 72시간의 완벽한 활용을 위해 최적화되어 있다.

### 절대 목표
- **핵심 성능**: Doosan M0609 + 2F 그리퍼로 물 따르기 ±1ml 정밀도 달성
- **완성도**: 실제 사용 가능한 시스템 구축 (프로토타입 수준이 아닌)
- **확장성**: 향후 다양한 액체 및 용기로 확장 가능한 아키텍처
- **안전성**: 상업적 환경에서도 사용 가능한 안전 시스템

### 성공 지표
- **정확도**: 50ml 목표량에서 95% 이상 ±1ml 달성
- **재현성**: 동일 조건 100회 반복 시 CV < 2%
- **신뢰성**: 8시간 연속 운영 중 예상치 못한 중단 0회
- **사용성**: 신규 사용자 10분 내 기본 작업 수행 가능

## 📅 Day 1: 시스템 기초 구축 및 첫 성공 달성

### 🌅 Morning Phase (Hour 0-8): 기반 시스템 완성

#### **Hour 0-2: 프로젝트 환경 완전 구축**

**작업 공간 설계**
2.5m × 2m × 1.5m 전용 작업 영역을 다음과 같이 구성:
- **로봇 베이스 영역**: 중앙 위치, 진동 격리 패드 설치
- **소스 스테이션**: 로봇 좌측 300mm 지점, 1L 물통 고정 거치대
- **타겟 스테이션**: 로봇 전방 400mm, 5개 타겟 용기 위치 마킹
- **안전 구역**: 작업 영역 외곽 200mm 완충 구역, 유출 방지 트레이
- **측정 스테이션**: 정밀 저울 전용 진동 격리 테이블

**정밀 측정 시스템 설치**
- **기준 저울**: Mettler Toledo 0.1mg 해상도, 실시간 데이터 연동
- **환경 센서**: 온도/습도 모니터링, ±0.1°C 정확도
- **타이밍 시스템**: 마이크로초 단위 작업 시간 측정
- **데이터 로깅**: 모든 센서 데이터 1kHz 샘플링 및 저장

**안전 시스템 완전 구축**
- **물리적 차단벽**: 로봇 작업 영역 3면 투명 아크릴 설치
- **비상 정지**: 3개 위치 하드웨어 E-stop 버튼
- **유출 대응**: 즉시 흡수 가능한 고흡수성 매트 배치
- **전력 관리**: UPS 시스템으로 예상치 못한 전력 중단 방지

#### **Hour 2-4: 하드웨어 완전 캘리브레이션**

**로봇 기구학적 정밀 캘리브레이션**
Doosan M0609의 모든 관절에 대해 다음 수준의 정밀 캘리브레이션 수행:
- **절대 위치 정확도**: 각 관절별 ±0.05° 정확도 달성
- **반복 위치 정확도**: 100회 반복 이동 시 ±0.02mm 편차 내
- **온도 보상**: 작업 환경 온도 변화에 따른 기구학적 보정
- **동적 보정**: 다양한 속도에서의 위치 정확도 검증

**토크 센서 극한 정밀도 추출**
내장 6축 토크 센서에서 최대한의 정밀도를 추출:
- **영점 안정화**: 30분간 워밍업 후 모든 자세에서 영점 재설정
- **선형성 검증**: 1g 단위 분동으로 0-200g 범위 선형성 99.8% 이상
- **크로스토크 제거**: 축간 간섭 최소화를 위한 보정 매트릭스 구성
- **동적 응답**: 10Hz 이상 대역폭에서 안정적 토크 측정 보장

**환경 변수 완전 제어**
작업 환경의 모든 변수를 측정 가능하고 제어 가능한 수준으로 관리:
- **진동 격리**: 건물 진동이 토크 측정에 미치는 영향 1% 미만
- **기류 제어**: 에어컨 송풍이 액체 표면에 미치는 영향 차단
- **조명 안정화**: 일정한 조명으로 액체 표면 반사 패턴 안정화
- **전자기 노이즈**: 토크 센서 신호에 영향을 주는 전자기 간섭 차단

#### **Hour 4-6: 기본 소프트웨어 아키텍처 구현**

**실시간 제어 시스템 구축**
ROS2 기반 분산 아키텍처로 다음 노드들을 구성:

**Core Control Nodes**:
- **master_controller**: 전체 시퀀스 관리 및 조정
- **weight_estimator**: 토크 데이터를 실시간 무게로 변환
- **trajectory_executor**: 정밀 궤적 실행 및 모니터링
- **safety_supervisor**: 실시간 안전 감시 및 비상 대응

**Data Processing Nodes**:
- **sensor_fusion**: 다중 센서 데이터 융합 및 필터링
- **state_estimator**: 시스템 전체 상태 실시간 추정
- **performance_monitor**: 작업 성능 실시간 분석 및 기록
- **calibration_manager**: 동적 캘리브레이션 및 드리프트 보상

**User Interface Nodes**:
- **coordinate_input**: 사용자 좌표 입력 및 검증
- **status_display**: 실시간 시스템 상태 시각화
- **data_logger**: 모든 실험 데이터 구조화된 저장
- **report_generator**: 성능 리포트 자동 생성

**통신 및 동기화 시스템**
- **실시간 통신**: 제어 루프 간 1ms 이내 지연시간 보장
- **데이터 동기화**: 모든 센서 데이터 타임스탬프 마이크로초 정확도
- **장애 복구**: 네트워크 또는 노드 장애 시 자동 복구 메커니즘
- **성능 모니터링**: 시스템 부하 실시간 모니터링 및 최적화

#### **Hour 6-8: 기본 무게 감지 시스템 완성**

**토크-무게 변환 고정밀 모델**
복잡한 물리적 관계를 정확히 모델링:
- **중력장 보상**: 로봇 자세별 중력 벡터 영향 완전 제거
- **관성 보상**: 로봇 움직임에 따른 관성력 실시간 보상
- **온도 드리프트**: 센서 온도 변화에 따른 자동 보정
- **비선형성 보정**: 토크 센서의 비선형 특성 고차 다항식으로 보정

**실시간 무게 추정 알고리즘**
마이크로초 단위의 정밀한 무게 추정:
- **칼만 필터**: Extended Kalman Filter로 노이즈 제거 및 상태 추정
- **적응적 필터링**: 작업 조건에 따른 필터 파라미터 자동 조정
- **이상치 제거**: 통계적 방법으로 센서 이상치 실시간 제거
- **다중 검증**: 여러 축의 토크 정보를 종합한 무게 추정 신뢰도 향상

### 🌞 Afternoon Phase (Hour 8-16): 첫 성공 달성

#### **Hour 8-10: 기본 따르기 동작 구현**

**정밀 궤적 계획 시스템**
물리학 기반의 최적 궤적 생성:
- **충돌 회피**: 모든 이동 경로에서 장애물과의 안전 거리 보장
- **동역학 최적화**: 로봇의 동역학적 한계 내에서 최단 시간 궤적
- **진동 최소화**: 액체 슬로싱을 최소화하는 부드러운 가속도 프로파일
- **에너지 효율**: 관절 토크 최소화를 통한 정밀도 향상

**유체역학 기반 제어**
액체 거동의 물리적 특성을 정확히 반영:
- **토리첼리 법칙 적용**: 용기 기울기에 따른 유출률 정확한 예측
- **표면 장력 효과**: 소량 액체에서 중요한 표면 장력 영향 보정
- **점성 효과**: 물의 점성이 유출 패턴에 미치는 영향 모델링
- **베르누이 효과**: 유출 과정에서의 압력 변화 정확한 계산

#### **Hour 10-12: 첫 번째 성공 시도**

**표준 테스트 시나리오 정의**
모든 후속 테스트의 기준이 되는 표준 조건:
- **소스 용기**: 500ml 원통형 비커, 무게 45g, 물 400ml 충전
- **타겟 용기**: 100ml 원통형 비커, 무게 25g, 초기 빈 상태
- **목표량**: 50ml (중간 수준의 난이도)
- **환경 조건**: 온도 23±1°C, 습도 45±5%, 무풍 상태

**좌표 시스템 정밀 정의**
밀리미터 단위 정확도로 모든 위치 정의:
- **소스 파지 위치**: (250, -100, 180mm), 수직 파지
- **소스 대기 위치**: (250, -100, 250mm), 파지 후 이동
- **타겟 상부 위치**: (250, 200, 200mm), 따르기 시작점
- **따르기 위치**: (250, 200, 180mm), 실제 따르기 높이
- **완료 후 위치**: (250, 200, 250mm), 안전한 대기 위치

**첫 성공 달성 목표**
구체적이고 측정 가능한 성공 기준:
- **정확도**: 목표 50ml 대비 실제량 45-55ml (±5ml)
- **시간**: 전체 작업 완료 시간 120초 이내
- **안전성**: 유출 없이 완전한 작업 완료
- **재현성**: 5회 연속 시도에서 3회 이상 성공

#### **Hour 12-14: 성능 분석 및 첫 최적화**

**상세 성능 분석**
매 시도마다 다음 데이터를 정밀 분석:
- **무게 측정 정확도**: 실시간 무게 추정값과 실제 무게 오차 분석
- **시간 분석**: 각 작업 단계별 소요 시간 및 지연 원인 분석
- **궤적 정확도**: 계획된 궤적과 실제 실행 궤적의 편차 분석
- **시스템 안정성**: 센서 노이즈, 제어 진동, 예상치 못한 동작 분석

**알고리즘 첫 번째 최적화**
분석 결과를 바탕으로 즉시 개선 적용:
- **센서 필터 조정**: 노이즈 특성에 맞는 필터 파라미터 최적화
- **제어 게인 튜닝**: PID 제어기 파라미터 실험적 최적화
- **궤적 세밀 조정**: 슬로싱 최소화를 위한 가속도 프로파일 개선
- **타이밍 최적화**: 각 단계별 대기 시간 최적화

#### **Hour 14-16: 안전 시스템 통합 및 검증**

**다층 안전 시스템 구현**
실시간으로 작동하는 4단계 안전 체계:

**Level 1 - 센서 레벨 안전**:
- 토크 센서 이상치 실시간 감지 (평균값의 3시그마 초과)
- 관절 각도 한계 초과 감지 (안전 마진 5도 적용)
- 움직임 속도 이상 감지 (설정값의 150% 초과)

**Level 2 - 제어 레벨 안전**:
- 제어 루프 응답 지연 감지 (10ms 초과)
- 궤적 추종 오차 초과 감지 (2mm 초과)
- 예상치 못한 외력 감지 (설정 임계값 초과)

**Level 3 - 작업 레벨 안전**:
- 작업 영역 경계 위반 감지 (정의된 안전 구역 이탈)
- 과도한 기울기 감지 (70도 초과)
- 비정상적 무게 변화 감지 (예상 범위 벗어남)

**Level 4 - 시스템 레벨 안전**:
- 하드웨어 통신 장애 감지
- 소프트웨어 프로세스 이상 감지
- 사용자 비상 정지 신호 처리

**비상 대응 프로토콜**
각 안전 레벨별 구체적 대응 방안:
- **즉시 정지**: 0.1초 내 모든 움직임 중단
- **안전 자세**: 2초 내 사전 정의된 안전 자세로 이동
- **상황 보고**: 이상 상황의 원인 및 대응 과정 자동 기록
- **복구 절차**: 안전 확인 후 작업 재개를 위한 체크리스트

## 📅 Day 2: 정밀도 극한 추구 및 고급 알고리즘

### 🌅 Morning Phase (Hour 16-24): 고급 센서 융합

#### **Hour 16-18: Extended Kalman Filter 완전 구현**

**다차원 상태 벡터 설계**
로봇 액체 따르기 시스템의 완전한 상태를 나타내는 12차원 벡터:
- **위치 상태**: [x, y, z] - 엔드이펙터 절대 위치
- **자세 상태**: [rx, ry, rz] - 용기 기울기 각도
- **동역학 상태**: [vx, vy, vz] - 움직임 속도
- **물리 상태**: [mass, mass_rate, bias] - 액체량 및 변화율, 센서 바이어스

**동적 모델 정밀 구현**
시스템의 물리적 거동을 정확히 예측하는 수학적 모델:
- **운동 방정식**: 로봇 동역학과 액체 동역학의 결합 모델
- **센서 모델**: 토크 센서 출력과 실제 물리량 간의 정확한 관계
- **노이즈 모델**: 환경 노이즈와 센서 노이즈의 통계적 특성
- **불확실성 전파**: 측정 불확실성이 상태 추정에 미치는 영향

**적응적 필터 튜닝**
실시간으로 최적 성능을 유지하는 적응형 시스템:
- **프로세스 노이즈 적응**: 작업 조건에 따른 시스템 노이즈 자동 조정
- **측정 노이즈 적응**: 센서 신뢰도 실시간 평가 및 가중치 조정
- **수렴성 보장**: 필터 발산 방지를 위한 안정성 모니터링
- **계산 최적화**: 실시간 처리를 위한 수치적 안정성 보장

#### **Hour 18-20: 다중 자세 고정밀 보정**

**포괄적 자세 매핑**
로봇의 모든 가능한 작업 자세에 대한 정밀 보정:
- **관절 공간 분할**: 6차원 관절 공간을 10도 간격으로 격자화
- **중력 보상 매핑**: 각 자세별 중력 토크의 정확한 계산 및 저장
- **관성 보상 매핑**: 움직임에 따른 관성력의 영향 사전 계산
- **교차 검증**: 인접 자세 간의 연속성 및 일관성 검증

- Group D: 적응형 예측 정지 (유출률 기반 동적 조정)

**실험 조건**:
- 목표량: 30ml (정밀도가 중요한 소량)
- 소스: 300ml 비커 (물 250ml)
- 타겟: 50ml 비커

**측정 항목**:
- 각 그룹별 오버슈트 발생량
- 목표량 정확도
- 따르기 완료 시간
- 시스템 안정성 (진동, 흔들림)

**절차**:
1. 각 그룹당 25회 반복 실험
2. 오버슈트량 정밀 측정 (0.1ml 단위)
3. 통계적 유의성 검증 (t-test)

**성공 기준**:
- 오버슈트 50% 이상 감소
- 정확도 저하 없이 예측 정지 효과 달성

### 실험 3-2: 슬로싱 억제 궤적 효과 검증

#### **목적**
Zero Vibration Derivative 입력 성형과 최적화된 궤적이 액체 진동 억제에 미치는 효과 측정

#### **실험 설계**

- Trajectory D: ZVD 입력 성형 적용

**측정 조건**:
- 소스 용기: 500ml 비커 (물 400ml) - 슬로싱 관찰 용이
